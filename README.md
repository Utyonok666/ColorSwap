## English

### 🎮 Overview
A dynamic 2D platformer that balances hardcore precision with meditative flow. The core experience is built around **color-shifting mechanics** and how the player's color interacts with the world.

### 🧩 Core Mechanics
* **Color Interaction:** Navigate the level by switching your color state.
* **Phase Shifting:** Match your color with platforms of the same hue to turn them semi-transparent, allowing you to pass through safely.
* **World Elements:** Use teleports and bounce pads to maintain momentum and reach the goal.
* **"Juicy" Feel:** Polished physics featuring Mario-style variable jump height, squash & stretch animations, and high-velocity movement.

### 🛠 Technical Stack
* **Engine:** Unity 2022.3+ (URP)
* **Physics:** Custom 2D Controller with Rigidbody2D interpolation for smooth movement.
* **Juice System:** Procedural squash & stretch and tilt logic based on velocity and input.
* **Rendering:** 2D Sprite-based with color-phasing logic.

### 🕹️ Controls
* **A / D** — Move
* **Space** — Jump
* **Left Shift** — Sprint / Boost
* **G** — Change Player Color (Red, Yellow, Green)

## Русский

### 🎮 Описание
Динамичный 2D-платформер, сочетающий в себе элементы хардкора и медитативного прохождения. Геймплей построен на **механике смены цвета игрока** и его взаимодействии с окружающим миром.

### 🧩 Основные механики
* **Цветовое взаимодействие:** Прохождение уровня завязано на переключении цвета персонажа.
* **Синхронизация:** Совпадение цвета игрока с цветом платформы делает её "полупрозрачной", позволяя беспрепятственно проходить сквозь неё.
* **Элементы мира:** Используйте телепорты и батуты, чтобы сохранять инерцию и добраться до финиша.
* **"Сочная" физика:** Настроенная инерция, прыжки в стиле Марио, процедурные анимации сжатия и растяжения.

### 🛠 Технический стек
* **Движок:** Unity 2022.3+ (URP)
* **Физика:** Кастомный 2D контроллер с интерполяцией Rigidbody2D для максимальной плавности.
* **Система "Сочности":** Процедурный Squash & Stretch и наклон (tilt) в зависимости от скорости и ввода.
* **Рендеринг:** 2D спрайты с логикой фазового смещения цветов.
  
### 🕹️ Управление
* **A / D** — Движение
* **Space (Пробел)** — Прыжок
* **Left Shift** — Ускорение
* **G** — Смена цвета игрока (Красный, Желтый, Зеленый)
